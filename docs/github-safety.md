# GitHub Safety Rules

This repository is designed for a private GitHub portfolio project.

- Do not commit real customer PDFs, real output Excel files, real SKU sheets, or API keys.
- `.gitignore` excludes `.env`, generated spreadsheets, PDFs, storage, uploads, and outputs.
- Use only files under `samples/` for screenshots, tests, and portfolio demos.
- The OpenAI key is optional and should be provided locally through `.env`.
- Public portfolio pages should show architecture, screenshots, and sanitized metrics only.
